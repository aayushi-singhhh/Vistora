# Vistora AI

A Streamlit application for AI-powered brand identity generation and visualization.

## Features

- Brand identity generation using AI
- Color palette extraction and visualization
- Logo generation
- Modern, responsive UI
- Downloadable brand kits

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set up your OpenRouter API key in `.streamlit/secrets.toml`:
```toml
OPENROUTER_API_KEY = "your-openrouter-api-key"
```

3. Run the application:
```bash
streamlit run genai.py
```

## Usage

The application provides an intuitive interface for generating comprehensive brand identities including color palettes, typography recommendations, and logo designs.
